# 📊 Design and Analysis of Algorithms Lab

This repository contains implementations of various algorithms studied in the **Design and Analysis of Algorithms (DAA)** lab. The programs are written in C and demonstrate different algorithmic techniques along with their performance analysis.

## Overview

The goal of this repository is to provide a clear understanding of fundamental algorithms, their design techniques, and efficiency analysis. Each program is implemented in a simple and structured way for easy learning.

## Algorithms Covered

### 🔹 Searching Algorithms
- Linear Search
- Binary Search

### 🔹 Sorting Algorithms
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

### 🔹 Divide and Conquer
- Merge Sort
- Quick Sort
- Binary Search

### 🔹 Greedy Algorithms
- Knapsack Problem (Greedy Approach)
- Job Scheduling

### 🔹 Dynamic Programming
- Fibonacci (DP)
- Knapsack Problem
- Longest Common Subsequence (LCS)

### 🔹 Backtracking
- N-Queens Problem
- Sum of Subsets

### 🔹 Graph Algorithms
- Breadth First Search (BFS)
- Depth First Search (DFS)
- Dijkstra’s Algorithm
- Prim’s Algorithm
- Kruskal’s Algorithm
